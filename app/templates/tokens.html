Como usarlo

Ejemplo curl:

curl -X POST {{ request.base_url }}api/run \
  -H "X-API-Token: TU_TOKEN_AQUI" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://www.youtube.com/watch?v=ID",
    "n_clips": 6,
    "profile": "viral",
    "ai_highlights": true,
    "instructions": "Solo clips donde habla de tema X"
  }'

{
  "job_id": "abc123",
  "status": "queued",
  "url_status": "/api/job/abc123",
  "url_view": "/job/abc123"
}
